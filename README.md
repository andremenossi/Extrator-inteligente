
# AutoMed Extrator G.O

Este é um aplicativo de automação desktop (Windows) para extração de dados da Intranet SIGS.

## Como Baixar o Executável (.exe)

1. Vá até a aba **Actions** deste repositório.
2. Clique no fluxo **Build AutoMed EXE** mais recente (com ✅ verde).
3. Role até a parte inferior em **Artifacts**.
4. Clique em **AutoMed-Installer** para baixar o .zip contendo o instalador.

## Como Usar

1. Instale e abra o programa.
2. Clique em **"Abrir Intranet"** e faça seu login manualmente.
3. Use os botões **"Definir"** para ensinar ao robô onde está a tabela e o botão.
4. Clique em **"Iniciar Extração"**.
5. Ao final, salve o arquivo Excel gerado.

## Segurança

* O software roda localmente.
* Nenhuma senha é salva.
* Nenhum dado de paciente é enviado para a nuvem.
